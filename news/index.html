OpenLieroX News

OpenLieroX 0.58 rc5 released
Posted on 2018-07-30

A new stable release is out! Version 0.58 rc5 adds IPv6 support, shows the death count in games with infinite lives, lets you navigate all menus with the arrow keys for better gamepad support, and fixes desynced shots and the invisible laser in network games. New game options let you mirror the map and make bonuses indestructible. All player skins from the Quality Skins Pack are now included. Grab it from the downloads page.

OpenLieroX 0.59 beta10 available for testing
Posted on 2012-04-17

The 0.59 beta line brings support for programmable levels and mods (using LUA, based on Gusanos), special light and graphics effects, 3D sounds, sounds for game events, and a single player mode. It is a testing release – try it out and report any issues. Download the beta here.

OpenLieroX 0.58 rc1 released
Posted on 2010-01-21

A huge update: a more stable network layer, a usable dedicated server, IRC support, new game modes (Hide & Seek, Capture The Flag, Race / Team Race), high-resolution and infinite level support, an extended physics engine and much more. See the full changelog on the downloads page.

OpenLieroX 0.57 rc1 released
Posted on 2008-05-10

The first release candidate of the 0.57 series fixes most of the reported crashes and the big lags when hosting a server caused by wrong bandwidth handling. Download it now.
