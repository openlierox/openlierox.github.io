## OpenLieroX Forum

The OpenLieroX forum is a dynamic, database-backed discussion board. Static
GitHub Pages hosting cannot run forum software, so the forum is not part of
this site.

Reach the community and get help here:

- [LXAlliance Forum](http://www.lxalliance.net/forum/) &ndash; the active community forum and FileBase
- [OpenLieroX on SourceForge](https://sourceforge.net/projects/openlierox/) &ndash; bug tracker, feature requests and downloads
- [Help]({{ '/help/' | relative_url }}) &ndash; an overview of all the ways to get support
